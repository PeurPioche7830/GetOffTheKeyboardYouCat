# Get Off The Keyboard, You Cat!
Basically the mac app "cats lock" but for broke and macless people

Did your cat ever walk on your keyboard?
Did you ever wish you could stop it from using your computer for you without having to be here to get it off your keyboard with your own hands?
Well don't worry, "Get off the keyboard, you cat!" is here to lock your computer (and, if you want, scare away) your feline companion without spending a dime on a paid, closed source, and mac only app.

Don't take this program too seriously, it's basically a coding exercise for me to learn python and tkinter without AI.

You need Python and Tkinter to get the .py running.

For now, this app is Windows only but porting to linux should be easy when the app is in an actually useful state.
